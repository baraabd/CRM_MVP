# CRM MVP (Monorepo)

Monorepo for a CRM MVP containing:
- **API**: `apps/api`
- **MOBILE**: `apps/crm-mobile`
- **Shared package** (optional): `packages/shared`

## Tech Stack
- Node.js + Nest typescript(workspaces)
- Docker + Docker Compose
- Postgre, SQLITE Database
- GitHub Actions CI
